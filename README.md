# SecAI+ Course Project

## Overview
Online course, study materials, and practice exams for CompTIA SecurityAI+ (CY0-001).
Created by Kevin Heaney — CompTIA Security+ certified, enterprise firewall engineer, WashU Data Science certificate, SecAI+ certified.

## Folder Structure

```
00_Source_Material/       → Raw study notes, chat exports, original Anki deck (NEVER published)
01_Free_Tier/             → Lead magnets: public Anki deck, YouTube scripts, blog posts, podcast
02_Paid_StudyGuide/       → Study guide PDF + domain practice quizzes ($39-49)
03_Paid_Course/           → Full video course modules, slides, supplemental materials ($99-199)
04_Paid_PracticeExams/    → Timed practice exams with answer explanations ($19-39)
05_Marketing/             → Landing pages, email sequences, social media, branding
06_Business/              → Pricing strategy, platform setup notes, analytics tracking
```

## Revenue Model
- **Free:** Anki deck (email capture), YouTube, LinkedIn, podcast
- **Tier 1:** Study guide + practice quizzes (Gumroad)
- **Tier 2:** Full video course (Udemy + Teachable)
- **Tier 3:** Practice exams (Gumroad, standalone or bundled)
- **Recurring:** Monthly membership with new questions, community, live Q&A

## Key Platforms
- Email: ConvertKit
- Digital sales: Gumroad
- Course hosting: Udemy (marketplace) + Teachable (owned)
- Podcast: Spotify for Podcasters
- Social: LinkedIn (primary), YouTube (secondary), X (engagement)
